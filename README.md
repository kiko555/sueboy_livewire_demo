## 在未啟動 container 前就先完成 composer install

因為如果是已經有寫程式了，有些套件得先下載，不然程式一起動，就會發生各種相依套件無法啟動的問題

這個 $(pwd) 是指程式碼所在的位置

```
docker run --rm -v $(pwd):/app composer install
```
